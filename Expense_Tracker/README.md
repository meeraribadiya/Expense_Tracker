# Expense Tracker

A simple, local-first expense tracker built with HTML, CSS, and JavaScript. Add expenses, view a dashboard with totals, and store everything in your browser using LocalStorage.

## Features
- Add new expenses
- View a list of all expenses
- See total balance, income, and expense summaries
- Delete individual expenses
- 100% local (no backend required)
